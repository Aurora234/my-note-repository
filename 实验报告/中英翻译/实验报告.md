#### 构建tokenizer基类

基类中需要有四个类变量分别表示`<unk>, <pad>, <sos>, <eos>`，将它们设置为类变量是为了可以在其他位置直接访问，之后tokenizer本身的类方法就需要直接访问这四个token。

```python
unk_token = '<unk>'
pad_token = '<pad>'
sos_token = '<sos>'
eos_token = '<eos>'
```

基类还需要有以下1个构造方法和1个普通方法以及3个类方法

##### 1 构造方法

```python
def __init__(self, vocab_list: list):
    self.vocab_list = vocab_list
    self.vocab_size = len(vocab_list)
    self.word2index = {word: index for index, word in enumerate(vocab_list)}
    self.index2word = {index: word for index, word in enumerate(vocab_list)}

    self.unk_token_index = self.word2index[self.unk_token]
    self.pad_token_index = self.word2index[self.pad_token]

```

**实例属性：**

- sos_token_index：同上

##### 3 encode

```python
def encode(self, text: str, add_sos_eos=False):
    '''
    将语料id化
    :param text: 语料
    :return: id化的语料
    '''
    tokens = self.tokenize(text)
    if add_sos_eos:
        tokens = [self.sos_token] + tokens + [self.eos_token]
    return [self.word2index.get(token, self.unk_token_index) for token in tokens]
```

**功能：**将输入的文本切词后再id化

**参数：**

- text：需要进行切词id化的文本
- add_sos_eos：若为True则在id化后的序列添加`sos和eos`的id

**返回：**切词并id化后的id序列

##### 4 build_vocab 类方法

```python
@classmethod
def build_vocab(cls, sentences: list, vocab_path):
    '''
    构建并保存词表
    :param sentences: 语料列表
    :param vocab_path: 保存路径
    :return: None
    '''
    vocab_set = set()
    for sentence in tqdm(sentences, desc='Building vocabulary'):
        vocab_set.update(cls.tokenize(sentence))

    vocab_list = [cls.pad_token, cls.unk_token, cls.sos_token, cls.eos_token] + [token for token in vocab_set if
                                                                                 token.strip() != '']
    print(f"词表大小: {len(vocab_list)}")

    with open(vocab_path, 'w', encoding='utf-8') as f:
        f.write('\n'.join(vocab_list))
```

**功能：**根据给定的语料列表（sentences）构建词表，并保存到指定路径（vocab_path）

**参数：**

- sentences：原始语料列表，每一个元素都是一句原始语料 
- vocab_path：词表保存的位置

**阐释：**

1. 遍历sentences，将每一句语料都进行分词后加入词表集合vocab_set（自动去重）
2. 去除词表集合中的`\n`和空格，并将`<unk>, <pad>, <sos>, <eos>`分别作为词表的前四个token
3. 保存此表vocab_list到指定位置，形式为每一行是一个token；'\n'.join(vocab_list)表示用换行符 `\n` 将列表 `vocab_list` 中的每个元素连接起来，形成一个字符串，其中每个元素占一行。

##### 5 from_vocab 类方法

```python
@classmethod
def from_vocab(cls, vocab_path) -> 'JiebaTokenizer':

    从已有的词表构建JiebaTokenizer
    :param vocab_path: 词表所在路径
    :return: JiebaTokenizer

```

**功能：**从指定路径vocab_path读取词表文件，并由该词表构建tokenizer

**参数：**

**返回：**tokenizer

**解释：**读取词表，直接传入本类构造方法

#### 构造EnglishTokenizer

需要调用nltk包的TreebankWordTokenizer, TreebankWordDetokenizer。

TreebankWordTokenizer是一个按照 Penn Treebank 项目标准设计的英语分词器，它遵循特定的语言学规则来处理文本。

TreebankWordDetokenizer：`TreebankWordTokenizer` 的逆操作，尝试将分词后的 token 列表重新组合成可读的文本。通常，detokenize后的字符串可能与原始字符串不完全相同，但会尽量保持可读性和语法正确性。

复写tokenize类方法和decoder方法

```python
class EnglishTokenizer(BaseTokenizer):
    tokenizer = TreebankWordTokenizer()
    detokenizer = TreebankWordDetokenizer()

    @classmethod
    def tokenize(cls, text: str) -> list[str]:
        return cls.tokenizer.tokenize(text)

    def decode(self, indexes: list[int]) -> str:
        tokens = [self.index2word[index] for index in indexes]
        return self.detokenizer.detokenize(tokens)
```

tokenize：将输入的文本`text`进行分词

decode：输入token的id列表，先将其还原为token列表，再还原为自然语言。

### 4. model.py

```python
import math

import torch
from torch import nn
import config


class PositionalEncoding(nn.Module):
    def __init__(self, d_model, max_len=256):
        super().__init__()
        self.d_model = d_model
        self.max_len = max_len

        pos = torch.arange(0, self.max_len, dtype=torch.float).unsqueeze(1)  # [max_len, 1]
        _2i = torch.arange(0, self.d_model, step=2, dtype=torch.float) # [d_model/2]
        div_term = torch.pow(10000, _2i / self.d_model) # [d_model/2]

        sins = torch.sin(pos / div_term)  # [max_len, d_model/2]
        coss = torch.cos(pos / div_term)  # [max_len, d_model/2]

        pe = torch.zeros(self.max_len, self.d_model, dtype=torch.float)
        pe[:, 0::2] = sins
        pe[:, 1::2] = coss

        self.register_buffer('pe', pe)

    def forward(self, x):
        seq_len = x.size(1)
        return x + self.pe[:seq_len]

class TranslationModel(nn.Module):

    def __init__(self, zh_vocab_size, en_vocab_size, zh_padding_idx, en_padding_idx):
        super().__init__()
        self.zh_embedding = nn.Embedding(zh_vocab_size, config.DIM_MODEL, padding_idx=zh_padding_idx)
        self.en_embedding = nn.Embedding(en_vocab_size, config.DIM_MODEL, padding_idx=en_padding_idx)

        # 位置编码
        self.position_encoding = PositionalEncoding(config.MAX_SEQ_LENGTH, config.DIM_MODEL)

        self.transformer = nn.Transformer(d_model=config.DIM_MODEL,
                                          nhead=config.NUM_HEADS,
                                          num_encoder_layers=config.NUM_ENCODER_LAYERS,
                                          num_decoder_layers=config.NUM_DECODER_LAYERS,
                                          batch_first=True)
        self.linear = nn.Linear(config.DIM_MODEL, en_vocab_size)

    def forward(self, src, tgt, src_pad_mask=None, tgt_mask=None):
        memory = self.encode(src, src_pad_mask)
        output = self.decode(tgt=tgt, memory=memory, tgt_mask=tgt_mask, memory_pad_mask=src_pad_mask)
        return output

    def encode(self, src, src_pad_mask=None):
        embed = self.zh_embedding(src)

        embed = self.position_encoding(embed)

        memory = self.transformer.encoder(src=embed, src_key_padding_mask=src_pad_mask)

        return memory

    def decode(self, tgt, memory, tgt_mask=None, memory_pad_mask=None):
        embed = self.en_embedding(tgt)
        embed = self.position_encoding(embed)

        output = self.transformer.decoder(tgt=embed, memory=memory, tgt_mask=tgt_mask,
                                          memory_key_padding_mask=memory_pad_mask)
        outputs = self.linear(output)

        return outputs

```

#### PostinalEncoding类

位置编码层（官方实现版本）

##### **构造函数：**

**参数：**

- d_model：Transformer的d_model参数
- max_len：token列表的最大长度，表示最多处理的文本长度

**功能：**获取位置编码矩阵，并把矩阵放入缓存中。当调用`to()`时，会把模型的参数以及缓存中的数据放入GPU，其它参数不会放入GPU



#### TranslationModel模型

**模型结构：**

1. zh_embedding与en_embedding层。（padding_idx参数告诉模型pad的下标位置，在反向传播时不需要更新）
2. 位置编码层position_encoding
3. Transformer层transformer
4. 线性回归层linear

因为推理阶段逻辑与训练阶段逻辑不一样，所以需要分别实现encode方法和decode方法。在训练阶段调用forward方法，编码和解码一套流程走完；在推理阶段解码器需要进行自回归生成：调用一次编码器，多次解码器自回归生成。

**encode方法**：

**参数：**

- src：模型输入数据，需要被翻译的原文
- src_pad_mask：编码器pad掩码

**解释：**编码需要将原数据词嵌入，位置编码，在经过Transformer编码器，得到编码器输出memory（[batch_size, src_seq_len, d_model]）

**decode方法**：

**参数：**

- tgt：解码器的输入，以sos开始的目标序列
- memory：编码器的输入，将在交叉注意力层作为Key和Value
- tgt_mask：解码器输入的因果掩码，迫使当前token只能把注意力放在前面的token，模仿自回归生成
- memory_pad_mask：交叉注意力层的掩码，作用与src_pad_mask一样（二者本身就是相同的），作为“二次保险”存在，再次确保编码器的输出与pad无关。
- tgt_pad_mask：本应有该参数，作用是对解码器的pad掩码，类似于src_pad_mask，但由于tgt_mask部分替代了它的功能，并且我们并不关心pad位置对应的输出，所以在本任务中不严谨的话可以不传入

**forward方法**：

**解释：**一并调用encode方法和decode方法。训练时调用。



### 5. dataset.py

```python
import pandas as pd
import torch
from torch.nn.utils.rnn import pad_sequence
from torch.utils.data import Dataset, DataLoader
import config


# 1. 定义Dataset
class TranslationDataset(Dataset):
    def __init__(self, path):
        self.data = pd.read_json(path, lines=True, orient='records').to_dict(orient='records')

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        input_tensor = torch.tensor(self.data[idx]['zh'], dtype=torch.long)
        target_tensor = torch.tensor(self.data[idx]['en'], dtype=torch.long)
        return input_tensor, target_tensor


# 2 提供一个获取dataloader的方法

def collate_fn(batch):
    # batch: 二元组列表:[(input_tensor, target_tensor)]
    input_tensor = [item[0] for item in batch]
    target_tensor = [item[1] for item in batch]
    # pad_sequence填充pad，因为每一句文本长度不一
    input_tensor = pad_sequence(input_tensor, batch_first=True, padding_value=0, padding_side='right')
    target_tensor = pad_sequence(target_tensor, batch_first=True, padding_value=0, padding_side='right')
    return input_tensor, target_tensor


def get_dataloader(train=True):
    path = config.PROCESSED_DATA_DIR / ("train.jsonl" if train else "test.jsonl")
    dataset = TranslationDataset(path)
    dataloader = DataLoader(dataset=dataset, batch_size=config.BATCH_SIZE, shuffle=True, collate_fn=collate_fn)
    return dataloader


if __name__ == '__main__':
    train_dataloader = get_dataloader()
    test_dataloader = get_dataloader(False)
    print(len(train_dataloader))
    print(len(test_dataloader))
    for inputs, targets in train_dataloader:
        print(inputs.shape, targets.shape)
        break
```

##### collate_fn方法

由于原数据每个样本长短不一，所以在构建dataloader的时候需要进行padding处理。

**参数：batch**：是一个二元组的列表`[(input_tensor, target_tensor)]`，表示一批数据。

**阐释及功能：**在dataloader划分batch后作用，首先将所有的`input_tensor`和`target_tensor`提取出来，之后用pad_seqence方法进行padding（由于我的词表中pad的下标是0，所以填充值为0）



### 6. train.py

```python
import time

from torch.utils.tensorboard import SummaryWriter
from tqdm import tqdm

from dataset import get_dataloader
import config
import torch
from tokenizer import ChineseTokenizer, EnglishTokenizer
from model import TranslationModel


def train_one_epoch(model, dataloader, loss_fn, optimizer, device):
    loss_total = 0.0
    model.train()
    for inputs, targets in tqdm(dataloader, desc="Train Epoch"):
        encoder_inputs, targets = inputs.to(device), targets.to(
            device)  # [batch_size, src_seq_len], [batch_size, tgt_seq_len]
        decoder_inputs = targets[:, :-1]
        decoder_targets = targets[:, 1:] # [batch_size, seq_len], [batch_size, seq_len]

        # 前向传播
        src_pad_mask = (encoder_inputs == model.zh_embedding.padding_idx)
        tgt_mask = model.transformer.generate_square_subsequent_mask(decoder_inputs.size(1))
        decoder_outputs = model(encoder_inputs, decoder_inputs, src_pad_mask=src_pad_mask, tgt_mask=tgt_mask)
        # decoder_outputs:[batch_size, seq_len, en_vocab_size]

        loss = loss_fn(decoder_outputs.reshape(-1, decoder_outputs.shape[-1]), decoder_targets.reshape(-1))
        loss.backward()
        optimizer.step()
        optimizer.zero_grad()

        loss_total += loss.item()
    return loss_total / len(dataloader)


def train():
    # 1 设备
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    # 2 数据
    dataloader = get_dataloader()
    # 3 分词器
    zh_tokenizer = ChineseTokenizer.from_vocab(config.MODELS_DIR / 'zh_vocab.txt')
    en_tokenizer = EnglishTokenizer.from_vocab(config.MODELS_DIR / 'en_vocab.txt')

    # 4 模型
    model = TranslationModel(zh_tokenizer.vocab_size, en_tokenizer.vocab_size, zh_tokenizer.pad_token_index,
                             en_tokenizer.pad_token_index).to(device)
    model.load_state_dict(torch.load(config.MODELS_DIR / 'best.pt'))
    # 5 损失函数
    loss_fn = torch.nn.CrossEntropyLoss(ignore_index=en_tokenizer.pad_token_index)

    # 6 优化器
    optimizer = torch.optim.Adam(model.parameters(), lr=config.LEARNING_RATE)

    # 7 TensorBoard
    writer = SummaryWriter(log_dir=config.LOGS_DIR / time.strftime("%Y-%m-%d_%H-%M-%S"))

    best_loss = float('inf')
    for epoch in range(1, config.EPOCHS + 1):
        print(f"{'=' * 10} Epoch {epoch} {'=' * 10}")
        loss = train_one_epoch(model, dataloader, loss_fn, optimizer, device)
        print(f"Loss: {loss:.4f}")

        # 记录到TensorBoard
        writer.add_scalar('loss', loss, epoch)

        # 保存模型
        if loss < best_loss:
            best_loss = loss
            torch.save(model.state_dict(), config.MODELS_DIR / 'best.pt')
            print("保存模型")
    writer.close()


if __name__ == '__main__':
    train()

```

训练阶段，损失函数用CrossEntropyLoss，优化器用Adam

**train方法：**见注释，训练逻辑放在train_one_epoch方法中。

**train_one_epoch方法：**

处理decoder_inputs：输入要有sos，而训练目标不需要sos，所以需要切片得到解码器输入和目标序列

transformer.generate_square_subsequent_mask方法可以得到因果掩码，形状:[tgt_seq_len, tgt_seq_len]



### 7. predict.py

```python
import torch
import config
from model import TranslationModel
from tokenizer import ChineseTokenizer, EnglishTokenizer


def predict_batch(model, inputs, en_tokenizer):
    '''
    批量预测
    :param model:  模型
    :param inputs: 输入 [batch_size, seq_len]
    :param en_tokenizer: EnglishTokenizer
    :return: 预测结果 [[*,*,*], [*,*,*,*], [*,*]]
    '''
    model.eval()
    with torch.no_grad():
        # 编码
        src_pad_mask = (inputs == model.zh_embedding.padding_idx)
        memory = model.encode(inputs, src_pad_mask)  # [batch_size, hidden_size]

        # 解码
        batch_size = inputs.shape[0]
        device = inputs.device
        # 解码以sos开始
        decoder_input = torch.full((batch_size, 1), en_tokenizer.sos_token_index).to(device)  # [batch_size, 1]

        # 预测结果缓存
        generated = []

        # 用来记录每个样本是否已经生成结束符
        is_finished = torch.full([batch_size], False, dtype=torch.bool, device=device)

        # 自回归生成
        for i in range(config.MAX_SEQ_LENGTH):
            # 解码
            tgt_mask = model.transformer.generate_square_subsequent_mask(decoder_input.size(1))
            decoder_output = model.decode(decoder_input, memory, tgt_mask, src_pad_mask)
            # shape = [batch_size, seq_len, vocab_size]
            # 保存预测结果
            next_token_indexes = torch.argmax(decoder_output[:,-1,:], dim=-1, keepdim=True)  # [batch_size, 1]
            generated.append(next_token_indexes)

            # 更新输入(decoder_input)
            decoder_input = torch.cat([decoder_input, next_token_indexes], dim=-1)

            # 判断是否应该结束
            is_finished |= (next_token_indexes.squeeze(1) == en_tokenizer.eos_token_index)
            if is_finished.all():
                break

        # 处理预测结果
        # generated: [tensor([batch_size, 1])]
        generated_tensor = torch.cat(generated, dim=1)
        # generated_tensor.shape = [batch_size, seq_len]
        generated_list = generated_tensor.tolist()
        # generated_list：[[*,*,*,eos,*], [*,*,eos,*]]
        # 去掉eos以及后面的token id
        for index, sentence in enumerate(generated_list):
            if en_tokenizer.eos_token_index in sentence:
                eos_pos = sentence.index(en_tokenizer.eos_token_index)
                generated_list[index] = sentence[:eos_pos]
    return generated_list


def predict(text, model, zh_tokenizer: ChineseTokenizer, en_tokenizer: EnglishTokenizer, device):
    # 4 处理输入
    indexes = zh_tokenizer.encode(text)
    input_tensor = torch.tensor([indexes], dtype=torch.long).to(device)

    # 5 逻辑预测
    batch_result = predict_batch(model, input_tensor, en_tokenizer)
    # top5_tokens = [tokenizer.index2word[index] for index in top5_indexes_list[0]]
    return en_tokenizer.decode(batch_result[0])


def run_predict():
    # 准备各种资源
    # 1 设备
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

    # 2 词表
    zh_tokenizer = ChineseTokenizer.from_vocab(config.MODELS_DIR / 'zh_vocab.txt')
    en_tokenizer = EnglishTokenizer.from_vocab(config.MODELS_DIR / 'en_vocab.txt')
    print("词表加载完成")

    # 3 模型
    model = TranslationModel(zh_tokenizer.vocab_size, en_tokenizer.vocab_size, zh_tokenizer.pad_token_index,
                             en_tokenizer.pad_token_index).to(device)
    model.load_state_dict(torch.load(config.MODELS_DIR / 'best.pt'))
    print("模型加载完成")

    print("欢迎使用中英翻译模型（输入q或者quit退出)")
    # input_history = ''
    while True:
        user_input = input("中文：")
        if user_input in ('q', 'quit'):
            break
        if user_input.strip() == '':
            print("请输入内容：")
            continue
        # input_history += user_input
        # print(f"输入历史:{input_history}")
        result = predict(user_input, model, zh_tokenizer, en_tokenizer, device)
        print(f"英文：{result}")


if __name__ == '__main__':
    run_predict()


```

run_predict根据用户输入调用predict方法，predict又会调用predict_batch进行批量预测。predict_batch是模型推理逻辑

**run_predict方法：**见注释

**predict方法：**首先处理输入文本，将文本分词并id化，由于输入形状要求[batch_size, seq_len]，而只输入了一句话（一个样本），所以需要再套一层列表。之后调用predict_batch方法得到推理结果，并将推理结果从id序列解码为文本

**predict_batch方法：**预测一批数据。

编码器与训练阶段相同。

解码器自回归生成，批输入`decoder_input`以sos开始生成，每次取解码器生成的最后一个token加入`decoder_input`，

循环MAX_SEQ_LENGTH次（MAX_SEQ_LENGTH表示模型最大能处理的文本token长度）。

并非所有的样本都需要循环生成最大次数，若所有的样本都生成了eos，就需要提前结束循环。is_finished（[batch_size]）标记了批次内所有样本是否已经生成了eos。在每次自回归生成后检查最后一个token是否是eos，由此更新is_finished。

最后处理预测结果缓存generated，将所有生成的id拼接起来，并逐样本检查，去除eos，截取有效token的id。返回



生成文本选择策略：本案例使用贪心策略



### 8. evaluate.py

```python
import torch
from nltk.translate.bleu_score import corpus_bleu

import config
from model import TranslationModel
from dataset import get_dataloader
from predict import predict_batch
from tokenizer import ChineseTokenizer, EnglishTokenizer


def evaluate(model, test_dataloader, device, en_tokenizer):
    predictions = [] # [[*,*,*,*], [*,*,*,*], [*,*,*]]
    references = []  # # [[[*,*,*,*]], [[*,*,*,*]], [[*,*,*]]]

    for inputs, targets in test_dataloader:
        # input: [batch_size, seq_len]
        # target: [[sos,*,*,*,eos,pad,pad...], [sos,*,*,*,*,eos,pad,pad...]...]
        inputs, targets = inputs.to(device), targets.tolist()
        # batch_result: [[*,*,*,*], [*,*,*,*], [*,*,*]]
        batch_result = predict_batch(model, inputs, en_tokenizer)  # [batch_size]

        predictions.extend(batch_result)
        references.extend([[target[1:target.index(en_tokenizer.eos_token_index)]] for target in targets])
    return corpus_bleu(references, predictions)

def run_evaluate():
    # 1 设备
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')

    # 2 词表
    zh_tokenizer = ChineseTokenizer.from_vocab(config.MODELS_DIR / 'zh_vocab.txt')
    en_tokenizer = EnglishTokenizer.from_vocab(config.MODELS_DIR / 'en_vocab.txt')
    print("词表加载成功")

    # 3 模型
    model = TranslationModel(zh_tokenizer.vocab_size, en_tokenizer.vocab_size, zh_tokenizer.pad_token_index,
                             en_tokenizer.pad_token_index).to(device)
    model.load_state_dict(torch.load(config.MODELS_DIR / 'best.pt'))
    print("模型加载成功")

    # 4 数据集
    test_dataloader = get_dataloader(False)

    # 5 评估逻辑
    bleu = evaluate(model, test_dataloader, device, en_tokenizer)
    print("评估结果:")
    print(f"bleu: {bleu}")


if __name__ == '__main__':
    run_evaluate()

```

**run_evaluate方法：**见注释

**evaluate方法：**将生成的所有样本保存到predictions中，真实目标保存到references中，调用corpus_bleu计算分数。
