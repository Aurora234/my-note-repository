# 数列

**等差数列**

定义：从**第二项起**，每一项与它前一项的**差为同一个常数** d（公差）：$$a_{n+1} - a_n = d$$

通项公式：$$a_n = a_1 + (n-1)d$$

前 n 项和公式：

- 首尾平均 × 项数  $$S_n = \frac{n(a_1 + a_n)}{2}$$
- 用首项和公差表示  $$S_n = na_1 + \frac{n(n-1)}{2}d$$

---

**等比数列**

定义：从**第二项起**，每一项与它前一项的**比为同一个常数** q（公比，$$q\neq0,1$$）：$$\frac{a_{n+1}}{a_n} = q$$

通项公式：$$a_n = a_1 q^{n-1}$$

前 n 项和公式：

- 公比 $$q \neq 1$$：$$S_n = a_1 \cdot \frac{1 - q^n}{1 - q}
  = \frac{a_1 - a_n q}{1 - q}$$
- **公比$$q = 1$$**（所有项都等于 a1）：$$S_n = n a_1$$

和公式推导：

> $$\begin{aligned}
> S_n &= a_1 + a_1q + a_1q^2 + \dots + a_1q^{n-1} \\
> qS_n &= \phantom{a_1 + }\;a_1q + a_1q^2 + \dots + a_1q^{n-1} + a_1q^n
> \end{aligned}$$
>
> 两式**相减**：
>
> $$S_n - qS_n = a_1 - a_1 q^n \\
> S_n(1-q) = a_1(1-q^n) \\
> \Rightarrow S_n = a_1 \frac{1-q^n}{1-q}$$
