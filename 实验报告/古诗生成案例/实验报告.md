### 1. 数据预处理

古诗生成案例中使用字符级分词，把每一个字作为词向量。并且不考虑所有的标点符号和空格。构建`word2id`和`id2word`，以及把每一行古诗都id化。

1. **逐行读取文件，保存诗的内容**
   读取文件，利用set的特性去重。将所有的字符保存到`char_set`中，每一行拆分为字符列表后作为`poems`的一个元素，`poems`保存的是所有诗的字符列表。

   ```python
   char_set = set()  # 去重的字符集合
   poems = []
   # 1.1 逐行读取文件，保存诗的内容
   with open(file_path, 'r', encoding='utf-8') as f:
       for line in f:
           line = re.sub(r'[，。？！、：]', '', line).strip()
           char_set.update(list(line))
           poems.append(list(line))  # poems.shape=[N, line.size]
   ```

2. **构建词表**
   构建`word2id`和`id2word`。除了所有的诗的字符外还需要添加"<UNK>"元素，防止OOV情况。

   ```python
   id2word = list(char_set) + ["<UNK>"]
   word2id = {word: idx for idx, word in enumerate(id2word)}
   ```

3. **将诗句id化**
   `poems`是所有诗的字符列表，而想要计算机处理就需要把所有字符id化，将字符列表转换为字符对应的id列表`id_seqs`。

   ```python
   id_seqs = []  # shape= [N, line.size]
   for poem in poems:
       id_seq = [word2id.get(word) for word in poem]
       id_seqs.append(id_seq)
   return id2word, word2id, id_seqs
   ```

4. **返回结果**

   ```python
   return id2word, word2id, id_seqs
   ```

**数据预处理函数**`preprocess_poems`如下所示

```python
def preprocess_poems(file_path):
    char_set = set()  # 去重的字符集合
    poems = []
    # 1.1 逐行读取文件，保存诗的内容
    with open(file_path, 'r', encoding='utf-8') as f:
        for line in f:
            line = re.sub(r'[，。？！、：]', '', line).strip()
            char_set.update(list(line))
            poems.append(list(line))  # poems.shape=[N, line.size]
    # 1.2 构建词表
    id2word = list(char_set) + ["<UNK>"]
    word2id = {word: idx for idx, word in enumerate(id2word)}
    # 1.3 将诗句id化
    id_seqs = []  # shape= [N, line.size]

    for poem in poems:
        id_seq = [word2id.get(word) for word in poem]
        id_seqs.append(id_seq)
    return id2word, word2id, id_seqs
```

### 2. 构建Dataset

**参数解释**：

- id_seqs：上一步数据预处理的id化诗字符列表
- seq_len：训练时的序列长度，滑动窗口的大小

滑动窗口的步长为1，意思是每次仅预测一个字符。将每一行诗都切分为训练数据和真值

**总代码如下所示**

```python
class PoetryDataset(Dataset):
    def __init__(self, id_seqs, seq_len):
        self.seq_len = seq_len
        self.data = []  # shape=[N, 2, seq_len]
        for id_seq in id_seqs:
            for i in range(0, len(id_seq) - self.seq_len):
                self.data.append((
                    id_seq[i:i + self.seq_len],
                    id_seq[i + 1:i + self.seq_len + 1],
                ))

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        x = torch.LongTensor(self.data[idx][0])
        y = torch.LongTensor(self.data[idx][1])

        return x, y  # shape = seq_len


dataset = PoetryDataset(id_seqs, 16)
```

### 3.构建模型

**模型结构**：模型包含：

1. Embedding
2. RNN
3. Linear

共三层结构

**参数解释**：

- vocab_size：词典大小，即所有的包含的去重的字符的数量
- embedding_dim：词嵌入的维度，词向量的维度
- hidden_size：RNN隐藏层维度
- num_layers：RNN层数

**输出**

- output：分类结果
- hn：RNN的每一层的最后输出的隐藏状态，对本案例来说就是RNN输出的最后一个隐藏状态[num_layers * num_direction, batch_size, hidden_size]、

```python 
class PoetryRNN(nn.Module):
    def __init__(self, vocab_size, embedding_dim, hidden_size, num_layers=1):
        super().__init__()
        self.embed = nn.Embedding(vocab_size, embedding_dim)
        self.rnn = nn.RNN(embedding_dim, hidden_size, num_layers, batch_first=True)
        self.linear = nn.Linear(hidden_size, vocab_size)

    def forward(self, input, hx=None):
        '''
        :param input: [batch_size, seq_len]
        :param hx: [batch_size, hidden_size]
        :return:
        '''
        embed = self.embed(input)
        output, hn = self.rnn(embed, hx)
        output = self.linear(output)
        return output, hn


```

### 4. 训练函数

**参数解释**

- model：模型
- dataset：dataset
- lr：学习率
- epoch_num：训练轮次
- batch_size：批数据量
- device：torch.device('cuda')

**准备**：

- 损失函数：nn.CrossEntropyLoss()
- 优化器：optim.Adam(model.parameters(), lr=lr)
- 数据加载器：DataLoader(dataset, batch_size=batch_size, shuffle=True)

注意损失函数的维度要求，CrossEntropyLoss要求input=[N,C]，output=[N]，或input=[N,C,d1,d2...]，output=[N,d1,d2....]

```python
def train(model, dataset, lr, epoch_num, batch_size, device):
    model.to(device)
    model.train()
    loss = nn.CrossEntropyLoss()
    optimizer = optim.Adam(model.parameters(), lr=lr)
    dataloader = DataLoader(dataset, batch_size=batch_size, shuffle=True)
    for epoch in range(epoch_num):
        train_loss = 0
        for batch_idx, (x, y) in enumerate(dataloader):
            x, y = x.to(device), y.to(device)
            output, _ = model(x)
            loss_value = loss(output.transpose(1, 2), y)
            loss_value.backward()
            train_loss += loss_value.item()
            optimizer.step()
            optimizer.zero_grad()
            print(f"\r epoch:{epoch:0>2}[{'='*(int(((batch_idx+1)/len(dataloader))*50)):<50}]", end='')
        print(f"train loss: {train_loss:.4f}")
        # this_loss = train_loss / len(dataset)
        # print(f"train loss: {this_loss:.4f}")
```

### 5. 古诗生成函数

**参数解释**：

- start_token：传入的字符串，要预测的诗的前几个字符
- line_num：要输出的行数
- line_length：每一句诗的长度

**代码解释**：每次预测的一个字符加入到poem中，预测完一句就加标点符号，每行两句，循环的line_num行。
current_rest_len表示当前句中还需要预测几个字符。
input_seq表示输入到模型的id化字符变量[1, seq_len]，除了第一个字符预测需要完整的输入字符列表，之后只需要上一步预测出来的一个字符。
hidden包含了之前所有的序列信息，所以预测时只需要一个字符和hidden
output[:, -1, :]意思是取预测序列的最后一个字符

**随机性生成：**并不是每次都选择概率最大的预测字符，保证生成随机性

````python
def generate_poem(model, id2word, word2id, start_token : str, line_num, line_length=7, device=torch.device('cpu')):
    model.eval()
    model.to(device)
    current_rest_len = line_length
    poem = [char for char in start_token]
    # start_id = word2id.get(poem[0], word2id["<UNK>"])
    # if start_id != word2id["<UNK>"]:
    #     poem.append(chars[0])
    #     current_rest_len -= len()
    current_rest_len -= len(poem)

    input_seq = torch.tensor([word2id[ch] for ch in poem], dtype=torch.long).unsqueeze(0).to(device)
    hidden = None
    with torch.no_grad():
        for i in range(line_num):
            for interpunction in ("，","。\n"):
                while current_rest_len > 0:
                    output, hidden = model(input_seq, hidden) # output.shape=[N, L, vocab_size]
                    output = output[:, -1, :]
                    # prob = torch.softmax(output, dim=-1)
                    # next_id = torch.multinomial(prob, num_samples=1).item()
                    next_id = torch.argmax(torch.softmax(output, dim=-1)).item()
                    poem.append(id2word[next_id])
                    input_seq = torch.LongTensor([[next_id]]).to(device)
                    current_rest_len -= 1
                poem.append(interpunction)
                current_rest_len = line_length

    return "".join(poem)
````
