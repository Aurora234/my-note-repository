# TensorBoard

```python
from torch.utils.tensorboard import SummaryWriter
import time

# 打开Tensorboard写入字节流，传入日志文件所在目录
writer = SummaryWriter(log_dir=config.LOG_DIR/time.strftime('%Y-%m-%d__%H-%M-%S'))

# 写入日志文件
# 坐标y值：train_loss， 坐标x值：epoch
writer.add_scalar('loss', train_loss, epoch)

# 当你不需要写入后
writer.close()
```

在终端打开TensorBoard：` tensorboard --logdir NLP/1-input-method-rnn/logs`
`--logdir`后填写日志文件所在目录（相对于当前工作目录的相对路径）



# 文件解释

运行顺序：process-->train-->predict-->evaluate

- src

  - config：所有的超参数和文件路径

  - dataset：构造dataloader的函数

  - model：模型结构

  - tokenizer：所有的词表操作（分词，id化，词表相关属性）

  - process：预处理

  - train：训练

  - predict：测试

  - evaluate：评估

- models

  - best.pt：保存的模型参数
  - vocab.txt：词表（每一行就是一个词）

- logs：tensorBoard日志

- data

  - processed：预处理后的训练集和测试集
  - raw：原数据集
